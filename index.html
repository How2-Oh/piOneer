Pioneer

A slo-mo action cam for Raspberry Pi.

View project on GitHub

Borrowing a page from the Astro Pi design book, piOneer consists of a 3D-printed case that holds a Raspberry Pi, camera module, user interface, LiPo battery, and large microSD card. Throw in some LEDs, wire, and fasteners and you have a weekend construction project that you can take with you anywhere and record everything that happens to you at 90fps.

A construction tutorial is available in The MagPi, Issue #50: https://raspberrypi.org/magpi-issues/MagPi50.pdf.
